* Fourier記述子による曲線クラスタリング

このリポジトリでは、2次元曲線を Fourier 記述子に変換し、その特徴量を用いて曲線群をクラスタリングする実験を行います。

** できること

- 2次元曲線を Fourier 記述子に変換
- Fourier 記述子同士の距離を計算
- 正弦波・矩形波・ノコギリ波の3種類の曲線を生成
- 各種類の曲線に回転・平行移動を適用
- 曲線群を特徴量ベースで比較しやすくする
- 曲線群の全組み合わせで距離行列を作成
- 距離行列を使って階層クラスタリングを実施
- 目標クラスタ数になるしきい値を自動探索
- クラスタリング結果を matplotlib で色分け表示

** 実装の概要

*** 1. Fourier 記述子

各曲線を複素数列 z = x + i y とみなし、FFT を使って低周波成分を特徴量として抽出します。

- 平行移動の影響を軽減
- スケールを正規化
- 必要に応じて係数の絶対値のみを使用可能

*** 2. 曲線群の生成

正弦波・矩形波・ノコギリ波をそれぞれ生成します。

- 各タイプの曲線にランダムな回転を適用
- 各タイプの曲線にランダムな平行移動を適用
- ノコギリ波は周期を短くして、より細かくぎざぎざした形にする

*** 3. 距離行列の作成

生成した曲線群に対して、各曲線の Fourier 記述子を作成し、そのユークリッド距離を距離行列として計算します。

*** 4. 階層クラスタリング

距離行列をそのまま使って階層クラスタリングを行います。

- `n_clusters` を指定してクラスタ数を決めることも可能
- `distance_threshold` を指定して、事前にクラスタ数を決めずに分割することも可能
- `average` linkage をデフォルトとして採用

*** 5. クラスタ数3のしきい値探索

距離行列から、クラスタ数が3になるしきい値を探索できます。

- 候補しきい値を複数試す
- 目標クラスタ数に最も近い値を選ぶ
- 実際に3クラスタになる値が見つかればそれを採用

*** 6. 可視化

クラスタラベルごとに色を分けて曲線を描画します。

- 単一のクラスタリング結果を表示可能
- Fourier記述子ベースの分類結果を表示可能

** 実行例

#+begin_src python
from main import generate_various_curves, compute_all_pair_distances, find_distance_threshold_for_cluster_count

curves = generate_various_curves(num_each_type=3, seed=42, sawtooth_period=1.57079632679)
distance_matrix = compute_all_pair_distances(curves, num_coefficients=16)

threshold = find_distance_threshold_for_cluster_count(
    distance_matrix,
    target_cluster_count=3,
)
#+end_src

** 依存ライブラリ

- Python 3.12+
- numpy
- matplotlib
- scipy

** 補足

- Fourier 記述子は形状の比較に有用ですが、完全なアフィン不変ではありません
- 曲線数を増やすと全組み合わせ距離計算が重くなります
- 階層クラスタリングでは、距離しきい値の選び方で結果が変わります
- 曲線の生成方法や平行移動量、回転角、特徴量の取り方によってクラスタ結果は変化します
